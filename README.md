# Uranus Grid Trading Bot (Phemex USDT-M)

Специализированный торговый бот для отработки пампов на листингах. Ориентирован на связку **Upbit Announcements -> Phemex USDT Perpetual**.

## Принцип работы
1. **Мониторинг**: Модуль `UpbitLiveMonitor` сканирует API анонсов Upbit. Использует умную ротацию прокси (SmartProxyManager) для обхода лимитов (HTTP 429).
2. **Сигнал**: При обнаружении ключевых слов (Listing, Support) извлекается тикер монеты.
3. **Исполнение (Entry)**:
   - Бот мгновенно проверяет наличие монеты на Phemex.
   - Если монета новая и её нет в кэше — запрашивает спецификации (лоты, тики) и перезапускает WS-стрим.
   - Выставляет рыночный ордер на вход.
4. **Управление выходом (Exit Scenarios)**:
   - **Grid TP**: Сразу после входа расставляется сетка из 5 лимитных ордеров на продажу. Параметры сетки (проценты и объемы) зависят от 24h объема монеты (3 тира: <500K, 500K-2M, >2M).
   - **Stop-Loss**: Динамический. До первого тейка — фиксированный %, после первого тейка — безубыток (entry price).
   - **TTL Market Close**: Если позиция не закрылась по сетке за X секунд, бот выходит по рынку.

## Настройка (cfg.json)
- `upbit.proxies`: Список прокси. `null` означает локальный IP.
- `upbit.poll_interval_sec`: Частота опроса. Бот сам посчитает эффективный интервал исходя из кол-ва прокси.
- `exit.scenarios.grid_tp`: Настройка ведер для разных объемов ликвидности.

## Команды
- `python main.py` — запуск бота в рабочем режиме.
- `python API/PHEMEX/ticker.py` — тест тикеров и объемов (Top-20).
