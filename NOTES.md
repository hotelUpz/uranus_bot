# Шпаргалка по работе с Git

---

## 1. Стянуть удалённый репозиторий, полностью синхронизировать и переключиться на ключевую ветку

```bash
git fetch --all
git reset --hard origin/master       # ВНИМАНИЕ: удалит все локальные изменения!
git checkout ключевая_ветка
git pull
```

---

## 2. Силой вмерджить ветку в master с последующим удалением этой ветки

```bash
git checkout master
git merge --no-ff имя_ветки         # Обычный merge, если нужен "силовой" вариант:
# git merge -X theirs имя_ветки     # Приоритет изменений из ветки (используй с осторожностью)
git branch -d имя_ветки             # Удалить локальную ветку
git push origin --delete имя_ветки  # Удалить ветку на удалённом репозитории
git push                            # Запушить изменения в master
```
